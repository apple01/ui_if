#有关工作流各一览画面的各种IF的使用说明Sample:

#IF: 检索申请一览列表
Url：ndims/ui_if/workflow/apply_list
默认参数Json模板：ndims/ui_if/workflow/json/apply_list.json
使用该接口生成一览画面，并调用相关iAP工作流功能的的Sample:
src/main/public/ndims/ui_if/workflow/sample_apply_list.jsp
注：Sample的URL例：http://localhost:8080/imart/ndims/ui_if/workflow/sample_apply_list.jsp
